Methodology

How the Alpha defect detection model was built

01

Data Preprocessing

Median imputation for 12 columns with missing values. RobustScaler normalization (resistant to outliers). All transformations fit exclusively on training data to prevent leakage.

SimpleImputerRobustScalerNo Leakage
02

Feature Engineering

Extended 49 raw parameters to 74 features. Stage-wise statistics (mean, std per rolling stage), inter-stage deltas, global aggregates (IQR, skewness, kurtosis, CV).

Stage StatsΔ Inter-stage74 Features
03

Imbalance Handling

Severe class imbalance (4.9% defect rate). Cost-sensitive learning with 194× class weight on defect class. Applied across all 5 ensemble members simultaneously.

Class Weight 194×Cost-sensitive4.9% Defect Rate
04

Ensemble & Threshold Calibration

5-model probability averaging (RF + XGB + LGB + ET + GB). Threshold set below the minimum defect score on training data — mathematically guarantees Recall = 100%.

5-Model AvgMin-Score ThresholdRecall = 1.0
